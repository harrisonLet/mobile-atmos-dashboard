How to Use

No installation required — the dashboard runs in your browser.

  • Map Viewer — Select a campaign and date to view the GPS-tracked measurement route on an interactive map.
  • Analysis
    1. Select a campaign and date to view the time series, GPS track, and analysis plots for the full day's measurements.
    2. Use Plotly's built-in tools to zoom, pan, and scale data ranges on the time series plot (top right).
    3. Use the select tool to drag a horizontal range on the time series — the GPS track, scatter plot, and wind rose will update to show only the selected window.
