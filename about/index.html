Mobile Atmos Dashboard

A web-based dashboard for visualizing mobile atmospheric field measurements. Built at the University of Utah's Land-Atmosphere Interactions Research Group.

Using the Dashboard

No installation required — the dashboard runs in your browser. Currently hosted at home.chpc.utah.edu/~u1460207/dashboard/

  • Map Viewer — Select a campaign and date to view the GPS-tracked measurement route on an interactive map.
  • Analysis

    1. Select a campaign and date to view time series plot, gps track, and analysis plots for the full day's measurements.

    2. Use Plotly's built-in tools to zoom, pan, and scale data ranges on the time series plot (top right).

    3. Use the select tool to select a horizontal range of data to update the supporting plots.

Use the Campaign and Date dropdowns in the nav bar to switch between datasets on any page.

Data Format

Data is organized under data/ by campaign ID and date:

data/
└── <campaign>/
    └── YYYYMMDD/
        ├── YYYYMMDD.csv
        └── YYYYMMDD_map.html

Hosting or Adding Data

The dashboard is static HTML — no server-side code is required. It can be hosted on any web server that serves static files.
